# Markdown

## Table Formatting

Always format tables with proper alignment for raw `.md` readability:

```md
| Column One | Column Two | Column Three |
| ---------- | ---------- | ------------ |
| Data       | Data       | Data         |
| Longer     | Short      | Medium data  |
```

Rules:
- Pipes `|` must align vertically across all rows
- Separator dashes `---` match column width exactly
- Single space after opening `|` and before closing `|`
- Pad content with spaces to match widest cell in column
